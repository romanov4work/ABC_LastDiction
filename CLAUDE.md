## О проекте
создаем обучающую игру для детей(от 6 до 11 лет) для конкурса от сайта Учи.ру. 
Приложение должно помогать детям развиваться (любой фиджитал навык, например дикция).
Приложение должно быть kids-friendly.
мы разрабатываем игру Прокачай Речь
04.05-отправка проекта;
06.05-результат конкурса;
07.05-защита проекта;

## Технологический стек
- билд должен запускаться без спец по, фронтенд javascript, бэкенд может быть написан на любом языке программирования, кроме delphi, haskell, erlang, elixir, clojure, c, c++, rust и visualbasic. для разработки игры разрешены только движки cocos или unity со сборкой под webgl, но если разрабатывать без движка, проще было бы на js. разработка идет именно web игры, т.е. фокус на браузер на телефонах и пк.
- не добавляй новые библиотеки без разрешения. не усложняй архитектуру. комить и пуш в гитхаб, чтобы если надо я мог сделать git revert (попроси у меня доступ к гх).
- Этот файл CLAUDE.md автоматически загружается при старте сессии
- Обновляй файл после значительных изменений, чтобы слеюущим клодам (которые работают через другой апи) было легче вникнуть в процесс
- Веди список завершенных задач для истории проекта
- Клод помни, что я использую пять бесплатных апи ключей от провайдера Kiro и у меня постоянно сменяются ключи когда подходят лимиты в 200к токенов, тебе необходимо при приближении к лимитам обновлять CLAUDE.md чтобы следующему апи ключу было легче понять. А также каждый новый апи ключ должен обновлять часто этот файл и всегда держать его в памяти!
- помни, что для красоты дизайна мы используем плагин /plugins frontend-design

### Что нужно сделать дальше:

**Приоритет 1: Карта уровней**
- Создать UI карты с темами "Скороговорки" и "Озвучь персонажа"
- Уровни в виде домиков/кружков вдоль речки
- Система прогресса (открытые/закрытые уровни)
- Возможность выбора пути (после уровня 1 → уровень 2 или 3)

**Приоритет 2: Тестовый уровень со скороговоркой**
- Скороговорка: "От топота копыт пыль по полю летит"
- Озвучка детским голосом (Web Speech API или аудио)
- Кнопка записи голоса (использует функцию startRecording())
- Показ времени произнесения (чистое время без пауз)
- Оценка дикции в процентах
- Цели: дикция 80%, время 3 секунды
- Кнопки: "Следующий уровень", "Повторить"

**Приоритет 3: Backend для Whisper**
- Настроить Whisper локально на сервере
- API для отправки аудио и получения транскрипции
- Сравнение с эталонным текстом (% совпадения)

**Приоритет 4: Персонажи и анимации**
- Добавить персонажей-зверят (спросить у пользователя спрайты)
- Анимации для персонажей

### Технические решения:
- **Распознавание речи**: Whisper (локально на сервере, бесплатно)
- **Озвучка**: Web Speech API или предзаписанные аудио детским голосом
- **Запись голоса**: MediaRecorder API (браузер)
- **Анализ времени**: вычисление чистого времени речи (без пауз в начале/конце)
- **Оценка дикции**: сравнение транскрипции Whisper с эталонным текстом (% совпадения)
