## О проекте

Приложение должно помогать детям развиваться (любой фиджитал навык, например дикция).
Приложение должно быть kids-friendly.

### Как работает распознавание (v2.1.9):
1. Пользователь нажимает "Записать голос"
2. Web Speech API запускается (window.SpeechRecognition)
3. Кнопка меняется на "Завершить запись"
4. API распознает речь моментально
5. Пользователь нажимает "Завершить запись" ИЛИ ждет 5 сек тишины (таймаут)
6. Возвращает {text: "распознанный текст", success: true}
7. Считается дикция через calculateAccuracy() (сравнение слов)
