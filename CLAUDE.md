## О проекте
создаем обучающую игру для детей(от 6 до 11 лет) для конкурса от сайта Учи.ру. 
Приложение должно помогать детям развиваться (любой фиджитал навык, например дикция).
Приложение должно быть kids-friendly.
мы разрабатываем игру Прокачай Речь

### КРИТИЧЕСКИ ВАЖНО для следующего Клода:
**Backend НЕ на GitHub!**
- ffmpeg файлы (200+ MB) НЕ загружены на GitHub (превышают лимит 100MB)
- Backend работает ТОЛЬКО локально на компьютере пользователя
- Для работы игры нужно запустить:
  1. `python backend/server_simple.py` (запускает Whisper API на localhost:5000)
  2. `ngrok http 5000` (создает HTTPS туннель для GitHub Pages)
- Ngrok URL: https://pried-isolation-joystick.ngrok-free.dev
- Если игра не распознает речь - проверь что backend запущен!

**Структура backend:**
```
/backend
  server_simple.py - рабочий Flask сервер (используй этот!)
  server.py - старая версия с логами
  test_direct.py - тест Whisper без Flask
  requirements.txt - зависимости (openai-whisper, flask, flask-cors)
  ffmpeg-master-latest-win64-gpl/ - НЕ в git (в .gitignore)
```

**Как работает распознавание:**
1. Пользователь записывает голос (MediaRecorder API)
2. Frontend отправляет audioBlob на ngrok URL
3. Flask получает файл, сохраняет во временный .wav
4. Whisper base модель распознает (3-5 сек)
5. Возвращает {text: "распознанный текст", success: true}
6. Frontend показывает результат и считает дикцию
