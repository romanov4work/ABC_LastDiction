## О проекте
создаем обучающую игру для детей(от 6 до 11 лет) для конкурса от сайта Учи.ру. 
Приложение должно помогать детям развиваться (любой фиджитал навык, например дикция).
Приложение должно быть kids-friendly.
мы разрабатываем игру Прокачай Речь
04.05-отправка проекта;
06.05-результат конкурса;
07.05-защита проекта;

### Как работает распознавание (v2.1.9):
1. Пользователь нажимает "🎤 Записать голос"
2. Браузер запускает Web Speech API (window.SpeechRecognition)
3. Кнопка меняется на "⏹️ Завершить запись"
4. Пользователь произносит скороговорку
5. API распознает речь моментально (без задержек)
6. Пользователь нажимает "⏹️ Завершить запись" ИЛИ ждет 5 сек тишины (таймаут)
7. Возвращает текст: {text: "распознанный текст", success: true}
8. Frontend показывает результат
9. Считается дикция через calculateAccuracy() (сравнение слов)
