## О проекте
создаем обучающую игру для детей(от 6 до 11 лет) для конкурса от сайта Учи.ру. 
Приложение должно помогать детям развиваться (любой фиджитал навык, например дикция).
Приложение должно быть kids-friendly.

**Как работает распознавание (v1.8.2):**

**Web Speech API (основной):**
1. Пользователь нажимает "🎤 Записать голос"
2. Браузер запускает Web Speech API (window.SpeechRecognition)
3. Пользователь произносит скороговорку
4. API распознает речь моментально (без задержек)
5. Возвращает текст: {text: "распознанный текст", success: true}
6. Frontend показывает результат с префиксом [Web Speech API]
7. Считается дикция через calculateAccuracy() (сравнение слов)

**Whisper Small (резервный):**
1. Пользователь нажимает "🎤 Записать голос"
2. MediaRecorder API записывает аудио в audioBlob
3. Пользователь нажимает "⏹️ Остановить запись"
4. Frontend отправляет audioBlob на ngrok URL через sendToWhisper()
5. Flask получает файл, сохраняет во временный .wav
6. Whisper small модель распознает (~10 сек)
7. Возвращает {text: "распознанный текст", success: true}
8. Frontend показывает результат с префиксом [Whisper Small]
9. Считается дикция через calculateAccuracy()
